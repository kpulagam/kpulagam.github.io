DGRL: Declarative Governance for Retrieval-Augmented Generation Systems
Under review at IEEE Access, 2025
Addresses critical gap in enterprise LLM deployments by introducing a declarative language and runtime framework for encoding information governance policies in RAG systems. Enables fine-grained access control, auditability, and compliance enforcement without modifying underlying LLM infrastructure. Demonstrates practical feasibility through implementation in production RAG pipeline with minimal latency overhead (<15ms per query).

Key contributions: Declarative policy language design, RAG-aware governance runtime, audit trail generation, performance evaluation on production workloads.
